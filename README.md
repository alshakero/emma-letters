# Emma Letters

A colorful, TV-friendly kids game for GitHub Pages. Emma presses a physical keyboard, and the page jumps a YouTube alphabet song to the matching `A-Z` letter.

## Run locally

```bash
npm run build
npm run serve
```

Then open the printed local URL.

## Deploy to GitHub Pages

This project deploys with the `gh-pages` npm package.

```bash
npm run deploy
```

`npm run deploy` builds the static site and publishes `dist/` to the `gh-pages` branch.

In GitHub, set Pages to deploy from the `gh-pages` branch, root folder.
